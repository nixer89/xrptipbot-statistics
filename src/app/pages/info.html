This site is dedicated around statistics about the XRPTipBot. It uses the fast tipbot-api available here: Super Fast XRPTipBot API Feed

You are free to use this API as well. A documentation can be found here: API Doc
The API works with query parameter only! You can use all json parameters from the result also in the query to filter your results.
Additionally you can use from_date, to_date and limit.
Here is an example of getting all tips to GoodXrp between 10th of May and 31st of May.:
https://api.xrptipbot-stats.com/feed?to=GoodXrp&from_date=2019-05-10 00:00:00&to_date=2019-05-31 00:00:00

The API Documentation can be found here: API Doc
Please read the API Docs carefully to get some cool tips!

You can even try the API there by clicking on the "Try it out" button! :-)
Pro tip: You can also generate a direct link (for bookmarks) to each users statistics page (or your own) by appending "?user=USERNAME&network=NETWORK" to https://xrptipbot-stats.com/userstatistics. The website will then open with the preselected values.
Example: https://xrptipbot-stats.com/userstatistics?user=Hodor&network=twitter
