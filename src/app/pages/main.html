This site is dedicated around statistics about the XRPTipBot. It uses the fast tipbot-api available here: Super Fast XRPTipBot API Feed

You are free to use this API as well. A documentation will follow soon. Just as a hint:
The API works with query parameter only! You can use all json parameters from the result also in the query to filter your results.
Additionally you can use from_date, to_date and limit.
Here is an example of getting all tips to xrpcharities between 1st of february and 3rd of february.:
https://xrptipbot-api.siedentopf.xyz/feed?to=xrpcharities&from_date=2019-02-01&to_date=2019-02-04
